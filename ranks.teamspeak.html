Our Teamspeak uses a military style ranking of it's members. Currently we are using eight diferent ranks.

The following six ranks are Server Groups.

Captain

This rank has full control of the server.

1st Lieutenant

This rank can create, edit and delete channels as well as kick, move and ban lower ranked users.

2nd Lieutenant

This rank can create create channels, it can also edit and delete the channels it has created.
This rank can also kick and move lower ranked users across channels.

Staff Sergeant

This rank allows for the broadcast of messages in the server channel as well as permissions to download and upload to the file browser.
For the time being we are allowing this rank to create temporary channels.

Private

Default group for any newcomer, this rank can download from the file browser.

Recruit

Special demotion rank. This rank prohibits pokes and private chats with other users.

The following three ranks are Channel Groups.

Sergeant

This rank can create sub-channels within the channel as well as kick users from the channel and grant talk power.

Corporal

This rank can grant talk power.

Private

Default channel rank, no special permission.
